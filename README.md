# rust-cli-programs

## Implementing barebones Linux utilities in Rust

Utilies Implemented so far:

- **cat**
- **echo**

To be implemented:
- **head**
- **wc**
- **uniq**
- **find**
- **cut**
- **grep**
- **ecs_provisioner** - to run one-time container tasks with AWS ECS
### How to run

```
cd utilities_folder
cargo build
cargo run -- --help
```
